# Unity 2D Platformer

A single-player 2D platformer built using Unity.  
This project demonstrates core Unity workflows, scene management, and gameplay scripting.

## Features
- Player movement, jump, and collision handling
- Enemy AI and basic combat mechanics
- Level design using Unity Tilemaps
- Camera control and transitions
- ScriptableObject-based data handling

## Tech Stack
- Unity (2D)
- C#
- Unity Input System
- Tilemap & Cinemachine

## How to Run
1. Clone the repository
2. Open the project in Unity Hub
3. Open the main scene and press Play

## Notes
This project was used as a base and is being actively modified to explore and improve Unity gameplay systems.
